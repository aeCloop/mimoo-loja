# 🎁 Mimoo Personalizados

Loja virtual moderna para venda de produtos personalizados, desenvolvida com foco em experiência mobile, facilidade de gerenciamento e integração com WhatsApp.

---

## ✨ Sobre o Projeto

A Mimoo Personalizados é uma plataforma de e-commerce desenvolvida para permitir a venda e gerenciamento de produtos personalizados de forma simples, rápida e intuitiva.

A aplicação oferece uma experiência otimizada para dispositivos móveis, permitindo que clientes realizem pedidos facilmente e entrem em contato diretamente pelo WhatsApp.

---

# 🚀 Funcionalidades

## 👤 Área do Cliente

### Catálogo de Produtos

* Listagem de produtos
* Filtro por categorias
* Exibição de preços por lote
* Visualização de imagens dos produtos

### Carrinho de Compras

* Adicionar produtos ao carrinho
* Atualizar quantidades
* Remover itens
* Finalização rápida de pedidos

### Pedido via WhatsApp

* Geração automática da mensagem
* Redirecionamento para WhatsApp
* Compatibilidade com Android e iPhone

### Dados Salvos Localmente

* Salvamento opcional de dados do cliente
* Preenchimento automático em novos pedidos
* Armazenamento via LocalStorage

### Progressive Web App (PWA)

* Instalação da loja como aplicativo
* Funcionamento otimizado para dispositivos móveis
* Ícone personalizado da Mimoo

# 🗄️ Banco de Dados

A aplicação utiliza Supabase como backend.

## Principais Tabelas

### products

Armazena os produtos cadastrados.

### product_lots

Armazena lotes e preços.

### categories

Categorias da loja.

### orders

Pedidos realizados pelos clientes.

### banners

Banners promocionais.

### app_settings

Configurações gerais da loja.
